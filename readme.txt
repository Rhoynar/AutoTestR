
need node installed

to get result copy your .html files into this js folder
then on console

node server.js [filename.html] [java/robot/cs] [output_file_name_is_optional]

example

node server.js popup.html java mytestfile

--> will create mytestfile.java into tests folder

OR

node server.js popup.html java

--> will create popup.java into tests folder
